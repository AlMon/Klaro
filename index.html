Vitaro

Grid

Vitaro uses the Bulma grid system which is flexible and extremely simple to use. All you have to do is next elements with the .column class within parent elements with the .columns class. If you need offset or positioning, that's pretty straightforward as well, you only need to check out the documentation on Bulma's website.

Buttons

Vitaro has 6 button styles to choose from. In order to take advantage of them, you can use the .button class alongside the class for the button style you want to use in order to get these button styles.

Additionally, you can use the .button-shadow class to give it a subtle shadow effect which changes with each different button style. You can also use the .button-block class to make the button occupy most of the width available.

Cards

Cards are a way of displaying information inside a larger image. A good use for them is to create links to articles, but they can be used any way you like. As you might expect, Vitaro cards use the .card class. If you want the text inside your card to be black, just add the .card-black class to your div.

There are additional elements that work very well within Vitaro cards. They are h2, p, small as well as elements with the .button class. They will style themselves in a visually pleasant way as shown next.

Feature Box

In order to highlight a feature or bring attention to something visually, you can use the .feature-box class. It will turn your column into a box with beautiful shadow details that will mantain your users' attention.
